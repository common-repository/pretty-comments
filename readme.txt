=== Pretty Comments ===
Contributors: AndreaBelvedere
Donate link: http://www.andreabelvedere.com/
Tags: post, page, comment
Requires at least: 2.5.0
Tested up to: 2.9-rare
Stable tag: 1.0

Replaces the regular wordpress comment textarea with a simple wysiwyg editor, it uses the jWYSIWYG jQuery (http://code.google.com/p/jwysiwyg/)

== Description ==

Add some formatting capabilities to the comments textareas.

= Bug Fixes =

Non yet, but please report any on http://www.andreabelvedere.com/opensource/pretty-comments

== Installation ==

1. Upload `pretty-comments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
